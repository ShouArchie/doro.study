Course Description

ECE 222:

Computer organization. Memory units, control units, I/O operations. Assembly language programming, translation and loading. Arithmetic logic units. Computer case studies.

View requirements for ECE 222

Topic List

Pre-midterm

  • Number systems review
  • RISC-V instruction set, addressing modes, subroutine linkage
  • Assembly language translation, linking, loading
  • Memory-mapped I/O, polling, interrupts, DMA

Post-midterm

  • Processor design: datapath, control unit
  • Pipelining: hazards, stalling, data forwarding, performance
  • Memory technology: SRAM, DRAM, DIMM
  • Memory hierarchy: caches, secondary storage, virtual memory
  • Computer arithmetic

 

Learning Outcomes

By the end of this course students should be able to:
Design computer programs using RISC-V assembly language
Understand and program input-output operations
Describe the parts of a computer, the internal structure of a processor, and steps for instruction execution in a processor
Develop control units for a simple processor
Describe a basic pipelined processor’s operations and implications for performance
Understand the memory hierarchy including the role and operation of main memory, cache memories, and virtual memory

Missed Midterm

  • If the midterm is missed for a valid reason (self-declared illness on Quest, short-term absence declared on Quest, other reasons approved by instructor) the midterm weight will be moved to the final exam.  Since the final exam is usually not cumulative, students who have the midterm weight moved to the final exam must submit an 8-page double-spaced summary of pre-midterm content that the instructor judges to be satisfactory.  The summary does not count in the final grade but failure to submit a satisfactory summary will result in a course grade of INC (incomplete).

Lab

  • See the lab manual on Learn for instructions.
  • Labs are done in pairs.
  • Demos are done during your scheduled lab period.  There is a 100% penalty for late demos so be sure to contact the lab instructor if something prevents you from demoing during the scheduled lab period.
  • Reports are due within 48 hours of your scheduled lab period and there is a 25% per day (including weekends) late penalty.
  • AI tools such as Copilot and ChatGPT may not be used to complete the labs.
