Course Description

CHE 524:

Experiments on process dynamics, control, and simulation of processes. Time constant; step and frequency response; controller tuning; multivariable control strategies. Implementation using simulation systems, mainframe computer control, microcomputers.

Course description:

Experiments on process dynamics,control and simulation of processes. Time constant; step and frequency response; controller tuning; multivariable control strategies; model predictive control. Implementation using simulation systems, main frame computer control, microcomputers.

Grading:

The Laboratory reports and the short quiz will be marked by the teaching assistant and the course instructor.

 

Lecture Topics:

  • Traditional Methods for System Identification in Process control.
  • Model Predictive Control, MPC.
  • Internal Model Control, IMC
  • Digital Sampling, Filtering and Control(z-transform).
